Quick Shorten Endpoint

You can easily generate short links bypassing the dashboard using our `GET` HTTP API.

GET https://monetlink.online/api/v1/tools/api?api=[YOUR_API_KEY]&url=[DESTINATION_URL]&alias=[CUSTOM_ALIAS]

Parameters (Query Matrix)

  • api: (Required) Your cryptographic 32-character token.
  • url: (Required) The destination link you want to monetize. Must be URL Encoded.
  • alias: (Optional) Specific short-code. If taken, system will throw an error.
  • format: (Optional) Default is JSON. Set `format=text` to get raw URL as response.

Example JSON Output

{
  "status": "success",
  "shortenedUrl": "https://monetlink.online/testcode12"
}

Example Text Output (format=text)

https://monetlink.online/testcode12
